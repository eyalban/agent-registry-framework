Agent Card JSON

ERC-8004 registration file: name, description, image, services (A2A/MCP endpoints), x402 support, trust models. Validated by Zod schema.

Wrapper Contract

Calls canonical register(), receives NFT, transfers to user, stores discovery tags. Collects optional 0.001 ETH fee.

Identity NFT

ERC-721 token on canonical Identity Registry. Owner = registering wallet. tokenURI = IPFS link to agent card.
